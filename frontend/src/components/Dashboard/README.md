# 仪表板组件

本目录包含仪表板（Dashboard）相关的可视化组件。

## 组件说明

### ProductionChart.tsx
生产数据图表组件。

**功能：**
- 显示生产数据趋势图
- 支持多种图表类型（折线图、柱状图等）
- 支持时间范围筛选
- 响应式设计
- 支持导出数据

**使用示例：**
```typescript
<ProductionChart
  data={productionData}
  timeRange="7d"
  onTimeRangeChange={(range) => setTimeRange(range)}
/>
```

**依赖：**
- Recharts（图表库）
- TanStack Query（数据获取）

### YieldCard.tsx
产量统计卡片组件。

**功能：**
- 显示产量统计数据
- 包含当前产量、目标产量、完成率等
- 支持趋势指示（上升/下降）
- 响应式设计

**使用示例：**
```typescript
<YieldCard
  current={1000}
  target={1500}
  title="今日产量"
  unit="件"
/>
```

## 数据流

1. 从 API 获取生产数据（使用 TanStack Query）
2. 数据经过处理和格式化
3. 传递给可视化组件进行渲染
4. 用户交互（如时间范围选择）触发数据重新获取

## 图表配置

### 图表类型
- **折线图**：用于显示数据趋势
- **柱状图**：用于对比不同时间点的数据
- **面积图**：用于显示累积数据

### 时间范围
- **1d**：最近 1 天
- **7d**：最近 7 天
- **30d**：最近 30 天
- **90d**：最近 90 天
- **1y**：最近 1 年

### 数据指标
- **总产量**：指定时间范围内的总产量
- **平均产量**：平均每天/每小时产量
- **完成率**：实际产量与目标产量的比率
- **趋势**：与上一周期相比的增长/下降

## 设计原则

1. **数据可视化**：清晰展示数据，避免信息过载
2. **交互性**：支持用户交互，如悬停显示详情、点击筛选等
3. **响应式**：在不同屏幕尺寸下都能良好显示
4. **性能优化**：大数据量时使用虚拟滚动或数据采样
5. **可访问性**：为图表提供文本描述，支持屏幕阅读器

## 注意事项

1. **数据格式**：确保传入的数据格式正确
2. **空状态**：处理无数据的情况，显示友好的空状态
3. **加载状态**：数据加载时显示骨架屏或加载动画
4. **错误处理**：数据获取失败时显示错误提示
5. **国际化**：支持多语言，特别是日期和数字格式
