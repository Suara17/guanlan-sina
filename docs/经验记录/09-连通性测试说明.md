# 快速连通性测试

## 后端接口测试

新添加的接口: `GET /api/test/ping`

响应格式:
```json
{
  "status": "success",
  "message": "Backend is connected!",
  "timestamp": "2025-01-12T11:49:47Z",
  "environment": "development"
}
```

## 前端测试组件

已在 `frontend/src/routes/_layout/index.tsx` (Dashboard页面) 添加连通性测试组件：

- 使用 React Query 进行 API 调用
- 显示连接状态（成功/失败）
- 实时显示后端响应数据
- 错误处理和加载状态

## 测试步骤

1. 确保后端运行在 http://localhost:8000
2. 确保前端运行在 http://localhost:5173
3. 访问前端首页，应该能看到"本地开发环境状态"卡片
4. 如果连接成功，会显示绿色的成功信息和JSON响应
5. 如果连接失败，会显示红色的错误信息

## 代理配置

前端 Vite 配置已设置代理：
- `/api/*` → `http://localhost:8000/*`
- 自动去除 `/api` 前缀

所以前端调用 `/api/test/ping` 实际访问后端的 `/test/ping`
