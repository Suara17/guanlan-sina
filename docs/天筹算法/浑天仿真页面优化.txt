从实际制造业场景出发，分析两种行业的可视化需求：

  ---
  轻工业 vs 重工业可视化需求分析

  🏭 轻工业（设备布局优化）

  典型场景：电子组装、服装制造、食品加工、精密仪器

  核心优化目标：减少物料搬运距离、提高空间利用率

  ┌──────────────┬─────────────────────────────────────────┬─────────────┐
  │  可视化元素  │                  说明                   │   优先级    │
  ├──────────────┼─────────────────────────────────────────┼─────────────┤
  │ 设备位置变化 │ 原始位置 → 优化后位置，带动画过渡       │ ⭐⭐⭐ 必须 │
  ├──────────────┼─────────────────────────────────────────┼─────────────┤
  │ 移动路径箭头 │ 显示设备移动轨迹和方向                  │ ⭐⭐⭐ 必须 │
  ├──────────────┼─────────────────────────────────────────┼─────────────┤
  │ 物料流路径   │ 设备之间的物料搬运路线（高频/低频区分） │ ⭐⭐ 高     │
  ├──────────────┼─────────────────────────────────────────┼─────────────┤
  │ 搬运距离对比 │ 优化前后的总搬运距离                    │ ⭐⭐ 高     │
  ├──────────────┼─────────────────────────────────────────┼─────────────┤
  │ 热力图       │ 高频作业区域标记                        │ ⭐ 可选     │
  ├──────────────┼─────────────────────────────────────────┼─────────────┤
  │ 车间边界     │ 可用空间范围                            │ ⭐⭐ 高     │
  └──────────────┴─────────────────────────────────────────┴─────────────┘

  关键指标：
  - 总移动距离减少量
  - 物料搬运成本降低
  - 空间利用率提升

  ---
  🚜 重工业（AGV路径优化）

  典型场景：汽车制造、钢铁冶炼、重工机械、船舶制造

  核心优化目标：AGV调度效率、避免路径冲突、提升吞吐量

  ┌──────────────┬──────────────────────────────────┬─────────────┐
  │  可视化元素  │               说明               │   优先级    │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 工位节点     │ 上料区、加工区、检测区、下料区等 │ ⭐⭐⭐ 必须 │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ AGV 实时位置 │ 多辆 AGV 的移动动画              │ ⭐⭐⭐ 必须 │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 行驶路径线   │ AGV 的规划路线（虚线/带方向）    │ ⭐⭐⭐ 必须 │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 时间轴控制   │ 显示调度进度，可拖拽回放         │ ⭐⭐ 高     │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 任务状态     │ 当前执行的任务（取料/运输/卸料） │ ⭐⭐ 高     │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 冲突检测点   │ 潜在碰撞区域高亮                 │ ⭐⭐ 高     │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 性能指标面板 │ 完工时间、瓶颈利用率、吞吐量     │ ⭐⭐⭐ 必须 │
  ├──────────────┼──────────────────────────────────┼─────────────┤
  │ 工位利用率   │ 每个工位的繁忙程度               │ ⭐ 可选     │
  └──────────────┴──────────────────────────────────┴─────────────┘

  关键指标：
  - 总完工时间
  - 瓶颈工位利用率
  - AGV 利用率
  - 路径冲突次数

  ---
  对比总结

  ┌──────────┬──────────────────────────┬──────────────────────────────┐
  │   维度   │          轻工业          │            重工业            │
  ├──────────┼──────────────────────────┼──────────────────────────────┤
  │ 核心动画 │ 设备移动（离散、一次性） │ AGV 移动（连续、循环）       │
  ├──────────┼──────────────────────────┼──────────────────────────────┤
  │ 路径展示 │ 设备搬迁路径             │ AGV 行驶轨迹                 │
  ├──────────┼──────────────────────────┼──────────────────────────────┤
  │ 时间维度 │ 静态布局对比             │ 动态调度过程                 │
  ├──────────┼──────────────────────────┼──────────────────────────────┤
  │ 交互需求 │ 切换对比（原始/优化）    │ 播放控制（暂停/倍速/进度条） │
  ├──────────┼──────────────────────────┼──────────────────────────────┤
  │ 指标重点 │ 搬运距离、空间利用       │ 完工时间、利用率             │
  └──────────┴──────────────────────────┴──────────────────────────────┘
