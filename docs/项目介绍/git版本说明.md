## 回退参考

如需回退到特定版本，请使用以下命令：

```bash
git reset --hard <commit-id>
```

例如：
- 回退到2026年1月底的稳定版本: `git reset --hard a124b46`
- 回退到登录系统完成版本: `git reset --hard a50ea0c`
- 回退到初始项目架构版本: `git reset --hard 5ac545c`
- 回退到项目初始提交: `git reset --hard 0d9caa3`
```

注意：硬重置会丢失当前工作目录的所有更改，请在执行前确保已备份重要数据。
