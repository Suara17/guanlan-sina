# 弈控经纬 系统操作说明

版本：2.0

更新日期：2026-02-26

---

## 概述

"弈控经纬"是一套面向离散制造业的"感知-推理-决策-执行"闭环智能操作系统。系统核心功能分为两条业务主线：

- **异常处理路线**：被动响应 — 从异常发现到方案执行的完整闭环
- **优化路线**：主动调度 — 基于工单切换的产线布局动态优化

---

## 一、异常处理路线（被动响应）

本路线解决工业现场的"突发异常处理"问题，展示系统从发现问题到量化决策的全过程。

### 第一步：异常触发与发现

**入口**：生产可视化看板（首页）

**操作步骤**：

1. 登录系统后进入生产可视化页面
2. 在页面左侧使用"产线选择器"切换目标产线（SMT产线 / PCB产线 / 3C组装线）
3. 观察页面右侧"异常信息"面板
4. 异常按严重程度分级显示：
   - **严重（Critical）**：红色标记，需立即处理
   - **高优（Error）**：橙色标记，优先处理
   - **告警（Warning）**：黄色标记，需关注

**界面说明**：
- 异常列表展示：发生时间、异常位置、异常描述、严重等级
- 右下角"司南数字人"会根据异常状态显示提醒信息
- 严重异常会触发语音/视觉告警

### 第二步：可解释归因分析

**入口**：点击异常列表中的具体异常条目

**操作步骤**：

1. 在异常列表中点击目标异常条目
2. 系统自动跳转至"格物知识图谱"页面
3. 查看异常的根因链路：
   - **现象节点**：异常的表面表现（如"连续出现10个锡焊不良"）
   - **直接原因**：导致现象的直接因素（如"焊锡温度偏低"）
   - **根本原因**：问题的根源（如"温控器老化，温度波动大"）
4. 点击图谱节点查看详细信息
5. 节点详情面板展示：
   - 节点类型（现象/原因/解决方案）
   - 详细描述
   - 关联关系

**核心价值**：系统不是给出黑盒结果，而是基于底层工业知识图谱，展示完整的"证据链条"。

### 第三步：量化智能决策

**入口**：在格物页面点击"智能决策"按钮，或从异常列表直接跳转至"司南智控"

**操作步骤**：

1. 进入司南智控页面后，查看根因分析结果
2. 系统自动展示多个候选处置方案
3. 每个方案附带量化的代价函数评估：

   | 评估维度 | 说明 |
   |---------|------|
   | 维修成本 | 预计维修所需的材料、人工费用 |
   | 交期影响 | 因异常导致的订单延误损失 |
   | 品质风险 | 选择该方案可能带来的质量风险成本 |
   | 停产损失 | 产线停机期间的产能损失 |
   | **综合期望损失** | 以上四项的加权综合评估 |

4. 使用排序功能切换排序维度：
   - 按成本排序：综合期望损失最低优先
   - 按时间排序：实施时间最短优先
   - 按风险排序：风险等级最低优先

5. 查看方案的详细信息：
   - 实施时间（小时）
   - 成功率（%）
   - 详细操作步骤

**核心价值**：系统不仅告诉用户"要做什么"，更能量化输出"模型评估与ROI对比报告"，支持高风险场景下的AI辅助决策。

### 第四步：闭环执行

**入口**：选择方案后点击"采纳并执行"按钮

**操作步骤**：

1. 确认选择的方案，点击"采纳并执行"
2. 系统自动跳转至"智行执行"页面
3. 查看执行任务信息：
   - 方案名称
   - 预计成本
   - 执行工单ID
4. 监控执行进度：
   - 当前执行步骤
   - 步骤完成状态
   - 预计剩余时间
5. 查看实时日志：
   - 执行动作记录
   - 设备状态变化
   - 告警信息
6. 执行控制操作：
   - 暂停执行（紧急情况）
   - 继续执行（恢复暂停）
   - 取消执行（放弃方案）
7. 执行完成后：
   - 系统显示"执行完成"确认
   - 结果回写系统进行复盘
   - 形成闭环，知识库更新

**执行完成后的操作**：
- 点击"返回诊断"：继续处理其他异常
- 点击"返回首页"：回到生产可视化看板

## 二、优化路线（主动调度）

本路线解决离散制造中"频繁换线导致产线拥堵"的问题，体现系统的复杂任务编排与空间运筹能力。

### 第一步：工单感知与预警

**入口**：生产可视化看板的"生产计划"模块

**操作步骤**：

1. 在生产可视化页面找到"生产计划"卡片
2. 查看当前工单信息：
   - 工单号（如：WO-20260224-001）
   - 产品编号与名称（如：PCB-A型）
   - 计划生产数量
   - 实际生产数量
   - 完成进度百分比
   - 预计完单时间

3. 查看下一工单预览：
   - 下一工单号
   - 下一产品编号与名称
   - 计划生产数量
   - 预计生产时长

4. 观察产品切换预警：
   - 当下一工单产品编号变化时，系统自动显示"产品变化预警"标签
   - 点击展开工艺流程对比，查看差异

### 第二步：工艺流程差异分析

**操作步骤**：

1. 点击当前工单的"工艺流程"折叠面板，查看当前产品的工序列表
2. 点击下一工单的"工艺流程"折叠面板，查看下一产品的工序列表
3. 系统自动标注差异：
   - **绿色（新增工序）**：下一产品新增的工序
   - **黄色（变更工序）**：工序参数发生变化的工序
   - **红色（移除工序）**：当前产品有但下一产品不需要的工序

4. 差异对比示例：
   ```
   当前产品 PCB-A：8道工序
   下一产品 PCB-B：12道工序
   
   差异说明：
   - 新增翻转工序（双面贴片）
   - 新增B面贴片工序
   - 新增二次回流焊工序
   - 新增ICT测试工序
   - 印刷周期时间增加5s
   ```

**核心价值**：系统内置每个产品的工艺流程数据，不同产品的工艺流程差异一目了然。

### 第三步：算法求解与方案生成

**入口**：点击"一键优化布局"按钮

**操作步骤**：

1. 确认产品切换信息后，点击"一键优化布局"按钮
2. 系统自动跳转至"天筹优化"页面
3. 系统自动创建优化任务，参数包括：
   - 行业类型（轻工/重工）
   - 车间尺寸
   - 设备数量
   - 可移动设备数
   - 固定设备数

4. 观察优化进度：
   - 任务状态（运行中/已完成/失败）
   - 进度百分比
   - 预计剩余时间

5. 优化完成后，查看结果：
   - 帕累托前沿三维可视化
   - 方案列表（含成本、工期、收益）

### 第四步：方案评估与选择

**操作步骤**：

1. 在帕累托前沿图表中查看方案分布
2. 点击方案查看详情：
   - 总成本
   - 实施工期
   - 预期收益
   - 技术指标（f1/f2/f3）

3. 使用决策偏好设置（可选）：
   - 调整成本权重
   - 调整工期权重
   - 调整收益权重
   - 点击"运行AHP决策"进行TOPSIS排名

4. 选择最优方案，点击"查看仿真效果"
5. 跳转至浑天仿真页面，查看布局优化效果：
   - 设备位置变化
   - 物流路径优化
   - AGV调度方案

## 三、异常模拟演示功能

本功能用于演示系统完整运行流程，便于向使用者展示系统能力。

### 操作步骤

1. 在生产可视化页面点击"异常模拟"按钮（紫色按钮）
2. 在弹出的场景选择器中选择模拟场景：
   - SMT线路焊接异常
   - 设备降速告警
   - 缺料预警
   - 其他预设场景（共10+情境）

3. 系统自动演示完整流程：
   - 异常触发 → 格物归因 → 司南决策 → 智行执行
   - 底层算法引擎接收预设参数
   - 前端完整展示"预警-归因-多方案对比"流程

### 演示数据说明

- 每个场景绑定具体参数（维修时间、成功率、预计成本等）
- 数据来源于预先收集的真实案例
- 演示流程与实际操作流程完全一致

## 四、其他功能模块

### 格物知识图谱

**入口**：侧边栏 → 格物知识图谱

**功能**：
- 展示异常、原因、解决方案之间的关系网络
- 支持按异常ID查看局部图谱
- 默认展示全量知识图谱
- 数据源支持Neo4j实时数据或Mock演示数据

### 浑天仿真验证

**入口**：侧边栏 → 浑天仿真，或从天筹优化跳转

**功能**：
- 产线/车间布局可视化
- AGV路径仿真
- 优化前后布局对比
- 指标变化展示

### 订阅价值评估

**入口**：侧边栏 → 订阅价值

**功能**：
- OEE提升统计
- 年度节省金额
- ROI投资回报率
- 月度趋势分析

### 能力商店

**入口**：侧边栏 → 能力商店

**功能**：
- 浏览模块化能力（视觉算法、优化引擎等）
- 订阅/开通新能力
- 查看已订阅能力

### OS内核接入

**入口**：侧边栏 → OS内核接入

**功能**：
- 扫描局域网设备（PLC、相机、传感器等）
- 协议匹配与接入确认
- 诊断报告生成

## 五、常见问题

| 问题 | 解决方案 |
|-----|---------|
| 无法登录 | 确认账号已创建且后端服务已启动 |
| 知识图谱为空 | 确认Neo4j服务已启动并导入数据，或使用Mock数据 |
| 优化任务无结果 | 检查天筹后端服务状态与任务参数 |
| 异常列表无数据 | 切换产线类型，或检查数据采集服务 |
| 执行任务卡住 | 查看实时日志，确认设备连接状态 |

## 六、数据来源说明

1. **知识图谱数据**：优先从Neo4j图数据库读取，不可用时降级到Mock演示数据
2. **生产数据**：当前部分数据为Mock演示数据，用于展示流程和交互
3. **优化算法**：后端调用真实多目标优化算法（SLP-GA等）
4. **执行日志**：实时从后端服务获取
